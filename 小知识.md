## 1
### input标签：type=password时，只对用户自己不可见，
### 解决方法：打开控制台，输入指令：
```
var div = document.getElementByTagName('input')[1];
div.value
```
####  即可显示出密码

##  4
### (1)行级元素、内联元素  inline
####  feature：内容决定元素所在位置
####           不可以通过css改变宽高
####  常见有：span strong em a del
### (2)块级元素  block
####  feature：独占一行
####           可以通过css改变宽高
####  常见有：div p ul li ol form address
### (3)行级块元素 inline-block
####  feature：内容决定元素所在位置
####           不可以通过css改变宽高
####  常见有：img
####  PS:默认css样式：(样式可修改)
```
span{
display:inline;
}
div{
display:block;
}
img{
display:inline-block;
}
```
